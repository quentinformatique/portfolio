Cuby

As a big fan of Rubik's cube, I wanted to create an app including a timer, a scramble generator and a statistics page and much more. I used Electron to create the app. The main goal of the project was to learn how to use Electron and to create a project from scratch about something I like. I wanted to make a nice looking app, so I spent a lot of time on the design. This project is still in development, but it helped me to learn a lot of things.
