ReadmeCreator

Eeach time I create a new project, I have to create a readme file. I wanted to automate this process, so I created a script that creates a readme file for me. I used python and tkinter to create a GUI. You just have to fill the fields and the script will create a readme file for you. The main goal of this project was to use tkinter and to create a script that can be useful for me.
